Windows安装wireshark没什么问题，主要说一下linux，先安装编译环境工具


然后进行安装：sudo apt-get install wireshark，结果在我的电脑上报错了：

原因在于我的电脑上有一个库文件的版本与wireshark需要的不同，再尝试安装wireshark-qt，结果继续报错，依次进行层层寻找，删除这个库

然后再安装就正常了
