Linux的启动过程
2020-03-11|Linux基础|
Word count: 150|Reading time: 1 min

几个重要文件的启动顺序：

  1. 通过/boot/vm进行启动 vmlinuz

  2. init /etc/inittab

  3. 启动相应的脚本，并且打开终端

  4. 启动login登录界面

  5. 登录，此时执行sh脚本的顺序，每次登录的时候都会完全执行的

这里就清楚了，如果不登录还要加载环境变量，就只能把环境变量放到rc.local里

添加开机启动项： sudo vim /etc/profile.d/apps-bin-path.sh， 在其中放入可执行文件
