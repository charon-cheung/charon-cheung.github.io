ROS2常用的命令
2024-04-25|ROSROS2|
Word count: 645|Reading time: 3 min

只记录ROS2 和 ROS1 不同的命令

pkg

topic

  • ros2 topic info topic_name

现在必须加/了

  • 输出话题的详细信息，包括发布订阅者: ros2 topic info /odom -v
  • ros2 interface show msg_name

显示 Topic 发送的消息定义 ros2 interface show sensor_msgs/msg/Imu

  • ros2 topic find msg_name

获知某个消息类型是谁在用

  • 手动发消息到话题

interface

  • ros2 interface list

分类显示消息、动作、服务的所有类型

  • ros2 interface package pkg_name

获知某个消息类型是谁在用，可以这样查

  • 显示消息的成员

service

  • ros2 service type

可以看到服务的定义。

  • ros2 service find

找出指定类型的所有服务

  • ros2 service find std_srvs/srv/Empty

  • ros2 interface show

显示服务的定义

param

  • ros2 param list
  • ros2 param describe
  • ros2 param get

  • ros2 param set (省略)

  • ros2 run + param

  • ros2 run —ros-args —params-file

rqt_graph 能够可视化节点和主题之间的连接。这个命令和ROS1一样，也是ROS2为数不多的GUI可视界面

tf

  • 可视化ROS2的tf树

ros2 run tf2_tools view_frames.py

  • ros2 run tf2_ros static_transform_publisher 1 2 3 0.5 0.1 -1.0 foo bar

  • ros2 run tf2_ros tf2_echo foo bar
