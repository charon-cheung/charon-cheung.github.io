使用web_video_server在网页显示图像

web_video_server是mjpeg_server的升级版，它的主要功能是，在ROS系统中，比如某些话题发布了图像、视频的话题，通过这个功能包可以订阅这些话题，然后对图像压缩，通过网络传输出去，这样我们就可以通过网络的方式获取到这些图像和话题。

安装：sudo apt-get install ros-indigo-web-video-server，默认的端口是8080，如果跟web服务器冲突了，需要修改端口。

web-video-server只有一个节点，名称也是web_video_server，而且有很多参数，所以launch文件更实用，建立的路径：先roscd web_video_server，然后mkdir launch，进入launch文件夹建立如下文件：

web_video_server中的address最好不要用127.0.0.1，而是用真实IP，否则只能在本机上有效，从其他电脑访问其IP不能成功．运行摄像头程序usb_cam后，运行此launch文件打开web_video_server，成功的话会看到这样的结果：

然后打开浏览器的地址http://ip_addr:8081，就会看到一个/node(topic)的链接，打开就是usb_cam中发布的图像，当然也可以在能ping通工控机的客户端上打开，但注意不要在翻墙状态。

URL的格式是http://ip:port/stream_viewer?topic=topic_name
