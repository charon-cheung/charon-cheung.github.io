strcmp

两个字符串的大小不能用关系运算符判断，应该用strcmp(a,b)，两个字符串相同时返回0，a>b返回一个正值,否则返回负值.
从左向右逐个比较a,b中的字符，根据二者的ASCII值确定结果，如果二者一直相同，但字符串a长度较小，那么结果为负。
我用的编译器对正负的处理是1和-1，有的是ASCII的差值。
